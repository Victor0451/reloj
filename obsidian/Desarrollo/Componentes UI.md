# Componentes UI

> [!info] Resumen
> Componentes disponibles en `src/components/ui/`.

---

## shadcn/ui Base

| Componente | Archivo | Uso |
|------------|---------|-----|
| Button | `button.tsx` | Botones con variantes: default, gradient, outline, ghost, destructive, link |
| Input | `input.tsx` | Campos de texto con hover + focus animado |
| Card | `card.tsx` | Contenedores con variantes: default, glass |
| Badge | `badge.tsx` | Etiquetas con variantes: default, success, warning, info, destructive, outline |
| Table | `table.tsx` | Tablas con zebra variant, padding aumentado |
| Dialog | `dialog.tsx` | Modales |
| Dropdown Menu | `dropdown-menu.tsx` | Menús desplegables |
| Avatar | `avatar.tsx` | Avatares con initials |
| Skeleton | `skeleton.tsx` | Loading placeholders |
| Sonner | `sonner.tsx` | Toast notifications con glassmorphism |
| Tooltip | `tooltip.tsx` | Tooltips |
| Separator | `separator.tsx` | Divisores |
| Label | `label.tsx` | Labels de formularios |
